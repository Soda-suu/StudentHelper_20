# StudentHelper_20
（学号尾号20）学生日常工具集合


## 项目介绍
面向学生的轻量化工具包，包含笔记模板、作业提醒清单等实用小功能。


## 如何使用
1. 直接下载仓库内的 `note-template.md`（笔记模板）；
2. 查看 `homework-reminder.md` 获取作业管理清单。


## 如何参与
- 提建议：在「Issues」里留言你的需求/想法；
- 改代码：新建分支（比如 `feature/xxx`），修改后提交PR。
